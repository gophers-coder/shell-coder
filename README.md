### 建议

- 简易的脚本使用 shell 脚本提升效率，复杂的优先考虑使用编程语言实现
- 减少使用 GUI
- 减少使用各种第三方插件等（原因：使用频率高了，容易忘记原生的命令，另外，服务器多是原生操作系统，插件少）
- 每天使用
- 每天创造机会使用
    - 开发环境选择：Linux, mac 等，Windows 暂不考虑（原因：服务器多以 linux 操作系统为主）
